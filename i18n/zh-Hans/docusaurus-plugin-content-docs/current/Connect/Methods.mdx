API 调用返回一个[`Promise`](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Promise)，resolve 状态时即能保证得到「结果」对象。

如果用户关闭窗口，或者网络连接时间如果失败，则将 `result.success` 设置为 false，将 `result.payload.error` 设置为错误信息，建议当遇到错误消息时建议记录错误信息并让用户重新开始操作。

每个方法都需要一个 [Object](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Object)，并带有 [`common`](./Method％20List/commonParams) 字段和方法特定字段。
