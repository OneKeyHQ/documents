## 获取代币信息

从 [coins.json](https://github.com/OneKeyHQ/connect/blob/onekey/src/data/coins.json) 文件中获取有关指定代币的信息。

ES6

```javascript
const result = await OneKeyConnect.getCoinInfo(params);
```

CommonJS

```javascript
OneKeyConnect.getCoinInfo(params).then(function (result) {

});
```

### 参数

[**可选的通用参数**](./commonParams)

#### 导出单个地址

* `coin` — *必填* `string` 代币符号（btc，eth，bch 等）。

### 例子

获取比特币的币种信息

```javascript
OneKeyConnect.getCoinInfo({
  coin: 'btc',
});
```
