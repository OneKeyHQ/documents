## 以太坊：验证消息

验证参数中的地址、签名、消息是否匹配。

ES6
```javascript
const result = await OneKeyConnect.ethereumVerifyMessage(params);
```

CommonJS
```javascript
OneKeyConnect.ethereumVerifyMessage(params).then(function(result) {

});
```

### 参数
[**可选的通用参数**](./commonParams)

[参数及类型](https://github.com/OneKeyHQ/connect/blob/onekey/src/js/types/params.js#L74-L78)

- `address` - *必填* `string` 签名人地址。"0x" 前缀是可选的
- `message` - *必须* `string` 签名的纯文本消息
- `hex`- *可选* `boolean` 转换来自十六进制的消息
- `signature` - *必须* `string` 十六进制格式的签名。"0x" 前缀是可选的

### 例子
```javascript
OneKeyConnect.ethereumVerifyMessage({
    address: "0xdA0b608bdb1a4A154325C854607c68950b4F1a34",
    message: "Example message",
    signature: "11dc86c631ef5d9388c5e245501d571b864af1a717cbbb3ca1f6dacbf330742957242aa52b36bbe7bb46dce6ff0ead0548cc5a5ce76d0aaed166fd40cb3fc6e51c",
});
```

### 结果

[参数及类型](https://github.com/OneKeyHQ/connect/blob/onekey/src/js/types/response.js#L133-L136)

```javascript
{
    success: true,
    payload: {
        message: "Message verified"
    }
}
```
错误
```javascript
{
    success: false,
    payload: {
        error: string // error message
    }
}
```
