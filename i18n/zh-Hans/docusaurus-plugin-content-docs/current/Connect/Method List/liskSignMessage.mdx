## Lisk：签署消息

要求设备使用由给定的 BIP32 路径派生的私钥对消息签名。

ES6
```javascript
const result = await OneKeyConnect.liskSignMessage(params);
```

CommonJS
```javascript
OneKeyConnect.liskSignMessage(params).then(function(result) {

});
```

### 参数

[**可选的通用参数**](./commonParams)

[参数及类型](https://github.com/OneKeyHQ/connect/blob/onekey/src/js/types/lisk.js#L109-L112)

- `path` - *必填* `string | Array<number>` 的最小长度为 3。 [了解更多](./path)
- `message` - *必填* `string` 消息，需要签署的消息纯文本。

### 示例

```javascript
OneKeyConnect.liskSignMessage({
    path: "m/44'/134'/0'",
    message: "example message"
});
```

### 结果

[参数及类型](https://github.com/OneKeyHQ/connect/blob/onekey/src/js/types/response.js#L104-L106)

```javascript
{
    success: true,
    payload: {
        publicKey: string,
        signature: string,
    }
}
```

错误
```javascript
{
    success: false,
    payload: {
        error: string // error message
    }
}
```
