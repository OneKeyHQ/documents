## Lisk：验证消息

在设备询问是否使用公钥和签名来验证消息。

ES6

```javascript
const result = await OneKeyConnect.liskVerifyMessage(params);
```

CommonJS

```javascript
OneKeyConnect.liskVerifyMessage(params).then(function (result) {

});
```

### 参数

[**可选的通用参数**](./commonParams)

[参数及类型](https://github.com/OneKeyHQ/connect/blob/onekey/src/js/types/lisk.js#L133-L137)

- `publicKey` - *必填* `string` 签名者公钥。
- `signature` - *必填* `string` 消息签名
- `message` - *必须* `string` 被签名的文本消息

### Example

```javascript
OneKeyConnect.ethereumVerifyMessage({
  publicKey: 'bf70027c9e4cea4584bd6016748c21e350708b2c166bf61ea78a147b5ff320ae',
  signature:
    'd39843f39983cf42609d1667f1c5a7958d8aef6b06880b93f67833630113a11c6847607a184d17da24bfaf799afc45fdcf2abef34142a23cabeb0d11374ac103',
  message: 'example message',
});
```

### 结果

[参数及类型](https://github.com/OneKeyHQ/connect/blob/onekey/src/js/types/lisk.js#L139-L142)

```javascript
{
    success: true,
    payload: {
        message: "Message verified"
    }
}
```

错误

```javascript
{
    success: false,
    payload: {
        error: string // error message
    }
}
```
