## Starcoin：验证消息

在设备询问是否使用公钥和签名来验证消息。

:::warning 重要提示
- 本方法自 OneKey Connect JS-SDK 版本 v8.6.0 之后支持，务必使用大于等于此版本的 npm 模块。
- 用户使用的 OneKey 固件也可能存在不兼容情况，务必处理好方法返回的报错内容。
:::

ES6

```javascript
const result = await OneKeyConnect.starcoinVerifyMessage(params);
```

CommonJS

```javascript
OneKeyConnect.starcoinVerifyMessage(params).then(function (result) {

});
```

### 参数

[**可选的通用参数**](./commonParams)

- `publicKey` - *必填* `string` 签名者公钥。
- `signature` - *必填* `string` 消息签名
- `message` - *必须* `string` 被签名的文本消息

### Example

```javascript
OneKeyConnect.ethereumVerifyMessage({
  publicKey: 'bf70027c9e4cea4584bd6016748c21e350708b2c166bf61ea78a147b5ff320ae',
  signature:
    'd39843f39983cf42609d1667f1c5a7958d8aef6b06880b93f67833630113a11c6847607a184d17da24bfaf799afc45fdcf2abef34142a23cabeb0d11374ac103',
  message: 'example message',
});
```

### 结果

```javascript
{
    success: true,
    payload: {
        message: "Message verified"
    }
}
```

错误

```javascript
{
    success: false,
    payload: {
        error: string // error message
    }
}
```

### 错误详情

1. 固件不支持时，返回结果。需提示用户升级至最新固件版本。
```json
{
  "success": false,
  "payload": {
    "error": "Unknown message",
    "code": "Failure_UnexpectedMessage"
  }
}
```
