## 签署消息

要求设备使用由给定的 BIP32 路径派生的私钥对消息签名。

ES6
```javascript
const result = await OneKeyConnect.signMessage(params);
```

CommonJS
```javascript
OneKeyConnect.signMessage(params).then(function(result) {

});
```

### 参数
[**可选的通用参数**](./commonParams)

[参数及类型](https://github.com/OneKeyHQ/connect/blob/onekey/src/js/types/params.js#L131-L135)

- `path` - *必填* `string | Array<number>` 的最小长度为 3。 [了解更多](./path)
- `message` - *强制* `string`
- `coin` - *可选* `string` 确定在 [coins.json](https://github.com/OneKeyHQ/connect/blob/onekey/src/data/coins.json) 中定义的币种。可以使用币种的字段包含 `coin_shortcut`，`coin_name` 或 `coin_label`。
- `hex` - *可选* `boolean` 转换来自十六进制的消息

### 示例
```javascript
OneKeyConnect.signMessage({
    path: "m/44'/0'/0'",
    message: "example message"
});
```

### 结果

[参数及类型](https://github.com/OneKeyHQ/connect/blob/onekey/src/js/types/response.js#L113-L116)

```javascript
{
    success: true,
    payload: {
        address: string,   // signer address
        signature: string, // signature in base64 format
    }
}
```

错误
```javascript
{
    success: false,
    payload: {
        error: string // error message
    }
}
```
