借助[MDX](https://mdxjs.com/)，您可以在Markdown中编写JSX并使用React组件。

`.mdx`扩展名不是必需的，但可以从工具（IDE，Prettier...）中获得更好的支持。

export const Highlight = ({children, color}) => (
    <span
        style={{
            backgroundColor: color,
            borderRadius: '2px',
            color: '#fff',
            padding: '0.2rem',
        }}
        onClick={() => alert('Highlight pressed!')}>
    {children}
  </span>
);

<Highlight color="#25c2a0">Docusaurus green</Highlight> and <Highlight color="#1877F2">
    Facebook blue
</Highlight> are my favorite colors.

我可以在 _JSX_ 旁边写**Markdown**！
