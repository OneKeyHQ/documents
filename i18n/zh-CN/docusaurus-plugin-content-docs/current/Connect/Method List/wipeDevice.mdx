## 擦除设备

将设备重置为出厂默认设置，并删除所有隐私数据。

ES6

```javascript
const result = await OneKeyConnect.wipeDevice(params);
```

CommonJS

```javascript
OneKeyConnect.wipeDevice(params).then(function (result) {

});
```

### 参数

[**可选的通用参数**](./commonParams)

- 通用参数 `useEmptyPassphrase` 始终为 `true`
- 通用参数 `allowSeedlessDevice` 始终为 `true`

### 示例

```javascript
OneKeyConnect.wipeDevice();
```

### 返回结果

正确返回

```javascript
{
    success: true,
    payload: {
        message: 'Device wiped'
    }
}
```

错误返回

```javascript
{
    success: false,
    payload: {
        error: string // error message
    }
}
```
