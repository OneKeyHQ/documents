## ripple：获取地址

在设备上显示请求的地址，并将其返回给调用者，向用户显示所请求密钥的描述，并要求其在 OneKey 上确认导出。

ES6
```javascript
const result = await OneKeyConnect.rippleGetAddress(params);
```

CommonJS
```javascript
OneKeyConnect.rippleGetAddress(params).then(function(result) {

});
```

### 参数
[**可选的通用参数**](./commonParams)

#### 导出单个地址

- `path` - *必填* `string | Array<number>` 的最小长度为 5。 [了解更多](./path)
- `address` - *可选* `string` 用于验证的地址（请参阅下面的**处理按钮请求**部分）
- `showOnTrezor` - *可选* `boolean` 决定是否在设备上显示地址。默认设置为 `true`

#### 导出多个地址

- `bundle` - `array` 具有 `path` 和 `showOnTrezor` 字段的对象

#### 处理按钮请求

可能处理 `UI.ADDRESS_VALIDATION` 事件，该事件一旦在设备上显示地址就会触发。您可以处理此事件并在应用程序内部显示自定义 UI。

如果满足某些条件，将完全不使用弹出窗口：
- 用户授予与 OneKey 通信的权限
- 设备通过 pin/passphrase 验证
- 应用程序已注册 `OneKeyConnect.on(UI.ADDRESS_VALIDATION，()=> {});`
- 设置参数 `address`
- 将参数 showOnTrezor 设置为 true (或完全不设置)
- 应用程序仅请求一个地址


### 示例

展示第二个 ripple 账户的第一个地址

```javascript
OneKeyConnect.rippleGetAddress({
    path: "m/44'/144'/1'/0/0"
});
```

展示多个 ripple 账户地址且无需在设备上确认

```javascript
OneKeyConnect.rippleGetAddress({
    bundle: [
        { path: "m/44'/144'/0'/0/0", showOnTrezor: false }, // account 1
        { path: "m/44'/144'/1'/0/1", showOnTrezor: false }, // account 2
        { path: "m/44'/144'/2'/0/2", showOnTrezor: false }  // account 3
    ]
});
```

使用自定义 UI 事件处理用户设备确认事件

```javascript
import OneKeyConnect, { UI } from '@onekeyhq/connect';

OneKeyConnect.on(UI.ADDRESS_VALIDATION, data => {
    console.log("Handle button request", data.address, data.serializedPath);
    // here you can display custom UI inside of your app
});

const result = await OneKeyConnect.rippleGetAddress({
    path: "m/44'/144'/0'/0/0",
    address: "rNaqKtKrMSwpwZSzRckPf7S96DkimjkF4H",
});
// dont forget to hide your custom UI after you get the result!
```

### 结果

仅展示一个地址的结果
```javascript
{
    success: true,
    payload: {
        address: string,
        path: Array<number>,
        serializedPath: string,
    }
}
```

展示多个地址的结果
```javascript
{
    success: true,
    payload: [
        { address: string, path: Array<number>, serializedPath: string }, // account 1, address 1
        { address: string, path: Array<number>, serializedPath: string }, // account 2, address 2
        { address: string, path: Array<number>, serializedPath: string }, // account 3, address 3
    ]
}
```

错误
```javascript
{
    success: false,
    payload: {
        error: string // error message
    }
}
```
