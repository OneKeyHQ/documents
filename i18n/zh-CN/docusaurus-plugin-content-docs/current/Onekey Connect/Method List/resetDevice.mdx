## 重置设备

恢复出厂设置并生成新的助记词。

ES6

```javascript
const result = await OneKeyConnect.resetDevice(params);
```

CommonJS

```javascript
OneKeyConnect.resetDevice(params).then(function (result) {});
```

### 参数

[\***\*可选的通用参数\*\***](./commonParams)

<br />

- `strength` — _可选_ `number` 可接受的值为[128 | 192 | 256]。 默认设置为 256
- `label` — _可选_ `string`
- `u2fCounter` - _可选_` number`。 默认值设置为当前时间戳(以秒为单位)。
- `pinProtection` - _可选_ `boolean`
- `passphraseProtection` - _可选_ `boolean`
- `skipBackup` - _可选_ `boolean`
- `noBackup` — _可选_ `boolean` 创建一个无种子设备

### 例子

```javascript
OneKeyConnect.resetDevice({
  label: 'My fancy Trezor',
});
```

### 结果

```javascript
{
    success: true,
    payload: {
        message: 'Device successfully initialized'
    }
}
```

错误

```javascript
{
    success: false,
    payload: {
        error: string // error message
    }
}
```
