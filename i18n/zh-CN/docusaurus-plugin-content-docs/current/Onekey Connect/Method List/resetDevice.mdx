## 重置设备
执行设备设置并生成新种子。

ES6
```javascript
const result = await OneKeyConnect.resetDevice(params);
```

CommonJS
```javascript
OneKeyConnect.resetDevice(params).then(function(result) {

});
```

### 参数
[****可选的通用参数****](./commonParams)
<br/>
*`strength` — *可选*`number`可接受的值为[128 | 192 | 256]。 默认设置为256
*`label` — *可选*`string`
*`u2fCounter`-*可选*`number`。 默认值设置为当前时间戳(以秒为单位)。
*`pinProtection`-*可选*`boolean`
*`passphraseProtection`-*可选*`boolean`
*`skipBackup`-*可选*`boolean`
*`noBackup` — —可选*`boolean`创建一个无种子设备

### Example
```javascript
OneKeyConnect.resetDevice({
    label: 'My fancy Trezor',
});
```

### Result
```javascript
{
    success: true,
    payload: {
        message: 'Device successfully initialized'
    }
}
```
Error
```javascript
{
    success: false,
    payload: {
        error: string // error message
    }
}
```
