## 获取地址

通过给定的 BIP32 派生路径，返回账户地址，同时可以在设备上显示该地址，并要求用户进行导出确认。

ES6

```javascript
const result = await OneKeyConnect.getAddress(params);
```

CommonJS

```javascript
OneKeyConnect.getAddress(params).then(function (result) {});
```

### 参数

[**可选的通用参数**](./commonParams)

#### 导出单个地址

- `path` - **必填** `string | Array<number>` 
  > 最小长度为 5。 [了解更多](./path)
- `address` - _可选_ `string` 
  > 用于验证的地址(请参阅下面的 `处理按钮请求` 部分)
- `showOnTrezor` - _可选_ `boolean` 
  > 是否在设备上显示地址和确认。默认值 `true`
- `coin` - _可选_ `string` 
  > 在[coins.json](https://github.com/OneKeyHQ/connect/blob/onekey/src/data/coins.json)中定义的币种。可以使用币种的字段包含 `coin_shortcut`，`coin_name` 或 `coin_label`。如果未指定 `coin` 参数，API 将尝试通过 `path` 参数自动获取币种。
- `crossChain` - _可选_ `boolean` 
  > 高级功能，仅当您了解该功能时开启。允许生成跨链地址，例如，莱特币网络上的比特币路径将以莱特币格式显示跨链地址。
- `multisig` - _可选_ 
  > 多签脚本类型，参考 [MultisigRedeemScriptType](https://github.com/OneKeyHQ/connect/blob/onekey/src/js/types/trezor.js#L107) (仅适用于 multisig 多签地址)
- `scriptType` - _可选_ 
  > 脚本类型，参考 [InputScriptType](https://github.com/OneKeyHQ/connect/blob/onekey/src/js/types/trezor.js#L113) 

#### 导出多个地址

- `bundle` - `Array` 
  > 包含 `path`，`showOnTrezor`，`coin` 和 `crossChain` 字段的对象数组

#### 处理按钮请求

从`onekey-connect@6.0.4`版本开始，新增 `UI.ADDRESS_VALIDATION` 事件，一旦设备上显示地址，该事件就会触发。您可以处理此事件并在应用程序里显示自定义 UI。

在以下情况下，将不会弹出确认窗口：

- 用户允许设备通信的权限
- 设备已通过密码验证
- 应用程序已注册 `OneKeyConnect.on(UI.ADDRESS_VALIDATION，()=> {});` 事件
- 提供了参数 `address`
- 参数 `showOnTrezor` 为 `true` (或未设置)
- 应用程序仅请求一个地址

### 示例

显示第一个比特币账户的第三个地址

```javascript
OneKeyConnect.getAddress({
  path: "m/49'/0'/0'/0/2",
  coin: 'btc',
});
```

批量获取第一个比特币账户的地址，且无需设备显示和确认

```javascript
OneKeyConnect.getAddress({
  bundle: [
    { path: "m/49'/0'/0'/0/0", showOnTrezor: false }, // address 1
    { path: "m/49'/0'/0'/0/1", showOnTrezor: false }, // address 2
    { path: "m/49'/0'/0'/0/2", showOnTrezor: false }, // address 3
  ],
});
```

在您的应用程序里，通过自定义界面验证地址

```javascript
import OneKeyConnect, { UI } from '@onekeyhq/connect';

OneKeyConnect.on(UI.ADDRESS_VALIDATION, (data) => {
  console.log('Handle button request', data.address, data.serializedPath);
  // 这里可以在您的应用中显示自定义 UI
});

const result = await OneKeyConnect.getAddress({
  path: "m/49'/0'/0'/0/0",
  address: '3L6TyTisPBmrDAj6RoKmDzNnj4eQi54gD2',
});
// 获取到结果后，记得关闭自定义 UI 窗口
```

### 返回结果

返回一个地址

```javascript
{
    success: true,
    payload: {
        address: string,     // displayed address
        path: Array<number>, // hardended path
        serializedPath: string,
    }
}
```

返回多个地址

```javascript
{
    success: true,
    payload: [
        { address: string, path: Array<number>, serializedPath: string }, // address 1
        { address: string, path: Array<number>, serializedPath: string }, // address 2
        { address: string, path: Array<number>, serializedPath: string }, // address 3
    ]
}
```

错误返回

```javascript
{
    success: false,
    payload: {
        error: string // error message
    }
}
```

### 旧版本迁移指南

v4 及以下版本

```javascript
OneKeyConnect.getAddress("m/49'/0'/4'/0/0", function (result) {
  // added "serializedPath" field
});
```

改成

```javascript
// params are key-value pairs inside Object
OneKeyConnect.getAddress({
    path: "m/49'/0'/4'/0/0"
}).then(function(result) {
    ...
})
```
