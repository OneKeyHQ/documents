## 擦拭设备
将设备重置为出厂默认设置，并删除所有私有数据。

ES6
```javascript
const result = await OneKeyConnect.wipeDevice(params);
```

CommonJS
```javascript
OneKeyConnect.wipeDevice(params).then(function(result) {

});
```

### 参数
[****可选的通用参数****](./commonParams)
<br/>
通用参数`useEmptyPassphrase`-设置为`true`
通用参数`allowSeedlessDevice`-设置为`true`

### Example
```javascript
OneKeyConnect.wipeDevice();
```

### Result
```javascript
{
    success: true,
    payload: {
        message: 'Device wiped'
    }
}
```
Error
```javascript
{
    success: false,
    payload: {
        error: string // error message
    }
}
```
