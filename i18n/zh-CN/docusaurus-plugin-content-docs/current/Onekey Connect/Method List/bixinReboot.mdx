## 将设备重新引导到引导加载程序
将设备重新引导至引导加载程序。

ES6
```javascript
const result = await OneKeyConnect.bixinReboot(params);
```

CommonJS
```javascript
OneKeyConnect.bixinReboot(params).then(function(result) {

});
```

### Example
```javascript
OneKeyConnect.bixinReboot();
```

### Result
```javascript
{
    success: true,
    payload: {
        message: 'device reboot started'
    }
}
```
Error
```javascript
{
    success: false,
    payload: {
        error: string // error message
    }
}
```
