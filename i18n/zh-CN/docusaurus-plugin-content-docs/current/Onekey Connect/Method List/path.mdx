## 小路
-`path`-`string | [BIP44](https://github.com/trezor/trezor-firmware/blob/master/core/docs/misc/coins-bip44-paths.md)路径方案中的Array <number>`或经过强化的“ Array” 数字。
### 例子
使用BIP44派生路径的比特币帐户1
```javascript
"m/49'/0/'0'"
```
使用BIP44派生路径的比特币帐户1
```javascript
[(49 | 0x80000000) >>> 0, (0 | 0x80000000 >>> 0), (0 | 0x80000000) >>> 0]
```
使用BIP44派生路径的帐户1的比特币第一地址地址
```javascript
"m/49'/0/'0'/0/0"
```
使用强化路径的帐户1的比特币第一地址地址
```javascript
[(49 | 0x80000000) >>> 0, (0 | 0x80000000) >>> 0, (0 | 0x80000000) >>> 0, 0, 0]
```
